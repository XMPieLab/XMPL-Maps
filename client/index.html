Full options object: 
{
	center: {
		address:'set address here'
	},
	zoom:8,
	width:'500px',
	height:'400px',
	route:{
		color: '#FF00FF', //color or route line
		origin:{ address : 'put origin address here'},
		destination: { address: 'put destination address here'}
	},
	routeIcons: {
		start: {
			url : 'url to icon image',
			size: { width: 30, height: 30},
			origin: {x:0, y:0},
			anchor: {x:0, y:0}
		},
		end: {
			url : 'url to icon image',
			size: { width: 30, height: 30},
			origin: {x:0, y:0},
			anchor: {x:0, y:0}
		}
	},
	pin: {
		address: 'put address here',
		icon: {
			url : 'url to icon image',
			size: { width: 30, height: 30},
			origin: {x:0, y:0},
			anchor: {x:0, y:0}
		}
	},
	geocoding_api_key: 'google api key'
}
Mandatory options:
  • center - If map will not display a route the this options is mandatory. When the map displays a route, the center will be overridden by the route.
  • geocoding_api_key - the api key to be used to fetch the address coordinates.
Default values:
  • height - 300px
  • center - latitude : 0, longitude : 0
